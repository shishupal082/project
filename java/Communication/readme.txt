This is a TCP IP java based communication program
It has 2 part
    - server
    - client
Both can be used independently

Command line arguments
    - 1: name (server/client, default server)
    - 2: protocol (byte / some other string, default byte)
    - 3: port (integer, default 9080)
    - 4: ip address (default 127.0.0.1)

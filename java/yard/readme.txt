To use this application
----------------------------

1) Double click on 'run_app.bat' file

2) Type following url in browser (Preferably chrome)

http://127.0.0.1:8080


Pre requisite
------------------
System should have installed java version-1.7 or above


Debugging if not working
--------------------------------
Type 'java' in command line it should display java Usage

If it is not displaying java Usage then try to install java in the computer
